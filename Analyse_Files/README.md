# AI Usage in Swiss Secondary Schools — Analysis Code

This repository contains the analysis code for the bachelor thesis on AI usage and perceptions among Swiss secondary school students.

## Structure
- `notebooks/` — 01_demographics_distribution, 02_AI_questions_distribution, 03_crosstabulations_demo_vs_AI_questions, 04_crosstabulations_AI_vs_AI_questions, 05_cluster_analysis
- `data/` — anonymised or synthetic samples only and the generated files with the test results
